CLASS LCL_BUS IMPLEMENTATION.
  METHOD CONSTRUCTOR.

    IF  IV_NUMBER IS INITIAL.

    ENDIF.

    IF IV_FARE IS INITIAL.
      RAISE NO_INPUT_FARE.
    ENDIF.

    MV_NUMBER = IV_NUMBER.
    MV_FARE = IV_FARE.

  ENDMETHOD.
