CLASS LCL_EVENT_HANDLER IMPLEMENTATION.
  METHOD ON_DOUBLE_CLICK.

    " 1. Local Stuctrue 선언 ( 제안한 이름 LS_SPFLI )

    "    Hint. READ TABLE 키워드와 INDEX 를 활용
    READ TABLE GT_SPFLI INTO LS_SPFLI INDEX ES_ROW_NO-ROW_ID.

    " 3. 이벤트 클래스의 정의에서 새로운 Private Instance Attribute인
    "    MT_SFLIGHT를 선언, 이 MT_SFLIGHT는 Line Type이 SFLIGHT로 이뤄진
    "    Internal Table ( Standard )

    " 4. Database Table SFLIGHT 에서 관련 데이터를 읽어와 MT_SFLIGHT에 보관한다.
    "    이때 Array Fetch ( INTO TABLE itab 문법을 사용하란 의미 )를 사용한다.
    "    단, MT_SFLIGHT에 보관되는 데이터는 Flight Connection과 동일한 키필드
    "    (내가 더블클릭한 라인의) 데이터로 제한한다.
    IF SY-SUBRC EQ 0.
      SELECT * FROM SFLIGHT INTO TABLE MT_SFLIGHT
       WHERE CARRID EQ LS_SPFLI-CARRID
         AND CONNID EQ LS_SPFLI-CONNID.

        IF SY-SUBRC EQ 0.
           IF MO_CONT_POPUP IS INITIAL.

             PERFORM CREATE_OBJECT_POPUP USING MO_CONT_POPUP.
             PERFORM CREATE_OBJECT_ALV USING MO_CONT_POPUP
                                             MO_ALV_POPUP
                                             .
             "CLOSE 이벤트 발생 했을 때 ON_CLOSE 실행을 위한 등록
             SET HANDLER ME->ON_CLOSE FOR MO_CONT_POPUP.

             PERFORM SET_TABLE_FOR_FRIST_DISPLAY USING MO_ALV_POPUP
                                                       MT_SFLIGHT.

           ELSE.
             CALL METHOD MO_ALV_POPUP->REFRESH_TABLE_DISPLAY
               EXCEPTIONS
                 FINISHED       = 1                " Display was Ended (by Export)
                 OTHERS         = 2.
           ENDIF.

        ENDIF.
    ENDIF.

    " 5. 디버깅 모드에서 MT_SFLIGHT가 상황에 알맞는 데이터를 보유하는지
    "    점검한다.

**    DATA LV_MSG TYPE STRING.

**    CONCATENATE '더블 클릭한 행 번호는' LV_ROW_ID

  ENDMETHOD.
