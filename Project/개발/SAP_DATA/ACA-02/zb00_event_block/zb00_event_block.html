REPORT zb00_event_block.

SELECTION-SCREEN FUNCTION KEY 1.

SELECTION-SCREEN BEGIN OF BLOCK b1 WITH FRAME TITLE TEXT-b01.
  PARAMETERS : pa_carr TYPE s_carrid OBLIGATORY.
SELECTION-SCREEN END OF BLOCK b1.

*&---------------------------------------------------------------------*
*& INITIALIZATION : selection screen 화면이 나오기 전에 행해지는 이벤트 (초기화,권한체크)
*&---------------------------------------------------------------------*
INITIALIZATION.
*  PERFORM check_auth.      "권한체크 로직
*  PERFORM initialization.  "초기값 셋팅 로직

*&---------------------------------------------------------------------*
*& AT SELECTION-SCREEN : selection screen '에서' 일어나는 이벤트
*&---------------------------------------------------------------------*
AT SELECTION-SCREEN OUTPUT. " 화면의 갱신 변경.... 모듈풀의 PBO
*  PERFORM modify_screen.   "화면에 대한 제어..

AT SELECTION-SCREEN ON pa_carr. "특정한 selection screen 포함된 필드에 제어
*  PERFORM check_carr USING pa_carr.

AT SELECTION-SCREEN ON VALUE-REQUEST FOR pa_carr. " F4를 설정해주는 이벤트
*  PERFORM f4_carr USING pa_carr.  " 특정 필드에 F4를 구성해주는 로직

AT SELECTION-SCREEN. "엔터 이벤트... 모듈풀의 PAI (기본형)

*&---------------------------------------------------------------------*
*& START-OF-SELECTION : 본격적인 데이터 처리 로직 (실행버튼 누르면 탄다..)
*&---------------------------------------------------------------------*
START-OF-SELECTION.
*  PERFORM get_data.   "데이터 추출
*  PERFORM make_data.  "데이터 가공


*&---------------------------------------------------------------------*
*& END-OF-SELECTION : 요즘은 자주 쓰지 않는다. 화면 출력 로직을 담당한다.
*&---------------------------------------------------------------------*
*END-OF-SELECTION.  " 생략 가능
*  PERFORM display_data.
*  CALL SCREEN 100.
