*&---------------------------------------------------------------------*
*& Report ZB03_FOR_ALL_ENTRIES
*&---------------------------------------------------------------------*
*&
*&---------------------------------------------------------------------*
REPORT zb03_for_all_entries MESSAGE-ID ZB03.
DATA : go_alv_grid TYPE REF TO cl_salv_table.

** 선언부
TABLES: ztsubject_b03, ztclass_b03.

" 출력하고자하는 테이블
TYPES: BEGIN OF gty_ent,
         sub_num   TYPE ztclass_b03-sub_num,
         sub_name  TYPE ztsubject_b03-sub_name,
         sub_close TYPE ztsubject_b03-sub_close,
       END OF gty_ENT.

DATA: gt_ent TYPE TABLE OF gty_ent,
      gs_ent TYPE gty_ent.

" 수강신청 인터널 테이블
DATA: gt_class TYPE table of ztclass_b03.

** 사용자 입력
SELECT-OPTIONS: so_sid FOR ztclass_b03-stu_id.

** 구현부
* 수강신청 인터널 테이블 조회
SELECT stu_id sub_num INTO CORRESPONDING FIELDS OF TABLE gt_class
  FROM ztclass_b03
  WHERE SUB_GRADE = 1 "검색 조건 1 - 1학년
  AND APPLY_SEMESTER = 2 " 2 - 2학기
  AND ztclass_b03~stu_id IN so_sid. "3 - 입력한 학생 ID

"정렬
SORT gt_class BY sub_num.
"중복 제거
DELETE ADJACENT DUPLICATES FROM gt_class COMPARING sub_num.

"데이터 여부 확인
IF gt_class IS NOT INITIAL.
  "인터널 테이블을 활용하여 데이터베이스 조회
  SELECT sub_num sub_name sub_close INTO TABLE gt_ent
    FROM ztsubject_b03 FOR ALL ENTRIES IN gt_class
    WHERE sub_num = gt_class-sub_num.

  "출력
  cl_salv_table=>factory(
  IMPORTING
    r_salv_table = go_alv_grid
    CHANGING
      t_table = gt_ent ).

  go_alv_grid->display( ).
ELSE.
 MESSAGE I001.
ENDIF.
