PARAMETERS: p_opt1 TYPE c RADIOBUTTON GROUP r1.
PARAMETERS: p_opt2 TYPE c RADIOBUTTON GROUP r1.
PARAMETERS: p_opt3 TYPE c RADIOBUTTON GROUP r1 DEFAULT 'X'. "조회 기본값-모든 항공

* FLTYPE 별 데이터 조회
CASE gc_c.
  WHEN p_opt1.
    SELECT * INTO CORRESPONDING FIELDS OF TABLE gt_spfli FROM spfli WHERE fltype = ' '.
  WHEN p_opt2.
    SELECT * INTO CORRESPONDING FIELDS OF TABLE gt_spfli FROM spfli WHERE fltype = 'X'.
  WHEN p_opt3.
    SELECT * INTO CORRESPONDING FIELDS OF TABLE gt_spfli FROM spfli.
ENDCASE.

** 출력부
PERFORM write_spfli USING gt_spfli.

*&---------------------------------------------------------------------*
*& Form write_spfli
*&---------------------------------------------------------------------*
*& 항공사 운항 일정 출력
*&---------------------------------------------------------------------*
*&      --> GT_SPFLI : 운항일정 테이블
*&---------------------------------------------------------------------*
FORM write_spfli USING pt_spfli like gt_spfli.
  DATA: ls_spfli LIKE LINE OF pt_spfli.

  LOOP AT pt_spfli INTO ls_spfli.
    WRITE: / ls_spfli-carrid, ls_spfli-connid,
             ls_spfli-cityfrom, ls_spfli-cityto,
             ls_spfli-deptime, ls_spfli-arrtime.
  ENDLOOP.
ENDFORM.
