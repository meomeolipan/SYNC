FUNCTION zfmb03_get_bezei.
*"----------------------------------------------------------------------
*"*"Local Interface:
*" IMPORTING
*" REFERENCE(IV_AUART) TYPE AUART
*" EXPORTING
*" REFERENCE(EV_BEZEI) TYPE TVAKT_BEZEI
*"----------------------------------------------------------------------
* Global data declarations

  DATA: lv_bezei TYPE tvakt-bezei.

    FROM tvakt
    WHERE auart = iv_auart  " 조건1 - AUART 코드와 일치
    AND spras EQ sy-langu.  " 조건2 - 시스템 언어와 동일
  ENDSELECT.

  ev_bezei = lv_bezei. "외부 파라미터에 조회 필드 할당

ENDFUNCTION.
