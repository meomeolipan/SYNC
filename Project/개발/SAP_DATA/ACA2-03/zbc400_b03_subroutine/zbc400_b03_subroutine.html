*&---------------------------------------------------------------------*
*& Form calc_percentage
*&---------------------------------------------------------------------*
*& Percentage 계산

*&      --> PA_INT1
*&      --> PA_INT2
*&      <-- GV_RESULT
*&---------------------------------------------------------------------*
FORM calc_percentage USING pv_act TYPE i
                               pv_max TYPE i
                      CHANGING cv_result TYPE tv_result.
  cv_result = ( pv_act / pv_max ) * 100.
